# Таблицы имён (Symbol Tables) в focal_lexer

Модуль для управления таблицами символов в лексическом анализаторе FOCAL.

## 📋 Структура

### Статические таблицы (инициализируются при создании SymbolTable):

1. **Таблица ключевых слов** - содержит команды языка FOCAL
2. **Таблица функций** - содержит встроенные математические функции
3. **Таблица разделителей** - содержит символы разделков
4. **Таблица операторов** - содержит математические и логические операторы

### Динамические таблицы (заполняются во время анализа):

5. **Таблица идентификаторов** - переменные, найденные в коде
6. **Таблица констант** - числовые константы, найденные в коде

## 🚀 Быстрый старт

### Базовое использование:

```python
from focal_lexer import SymbolTable

# Создание таблицы символов
symbol_table = SymbolTable()

# Добавление идентификаторов
id_cost = symbol_table.add_identifier('COST', 'Переменная с плавающей точкой')
id_price = symbol_table.add_identifier('PRICE', 'Переменная с плавающей точкой')
id_tax = symbol_table.add_identifier('TAX', 'Переменная с плавающей точкой')

# Добавление констант
c1 = symbol_table.add_constant('0.98', 'Константа с плавающей точкой')
c2 = symbol_table.add_constant('10.5', 'Константа с плавающей точкой')

# Вывод таблиц
symbol_table.print_all_tables()
```

### Интеграция с лексером:

```python
from focal_lexer import SymbolTable, LexicalAnalyzer, TokenType

symbol_table = SymbolTable()
lexer = LexicalAnalyzer(source_code)
tokens = lexer.tokenize()

# Собираем информацию в таблицы
for tok in tokens:
    if tok.type == TokenType.IDENTIFIER:
        symbol_table.add_identifier(tok.value)
    elif tok.type == TokenType.NUMBER:
        symbol_table.add_constant(tok.value)

# Выводим таблицы
symbol_table.print_identifiers_table()
symbol_table.print_constants_table()
```

## 🔧 API

### Класс SymbolTable

#### Методы добавления:
- `add_identifier(name: str, info: str = "Переменная") -> int` - добавить идентификатор
- `add_constant(value: str, const_type: str = "Константа с плавающей точкой") -> int` - добавить константу

#### Методы получения информации:
- `get_identifier(name: str) -> dict | None` - получить информацию об идентификаторе
- `get_constant(value: str) -> dict | None` - получить информацию о константе

#### Методы вывода:
- `print_keywords_table()` - вывести таблицу ключевых слов
- `print_functions_table()` - вывести таблицу функций
- `print_delimiters_table()` - вывести таблицу разделителей
- `print_operators_table()` - вывести таблицу операторов
- `print_identifiers_table()` - вывести таблицу идентификаторов
- `print_constants_table()` - вывести таблицу констант
- `print_all_tables()` - вывести все таблицы

## 🎯 Особенности

✅ **Уникальность элементов** - одинаковые идентификаторы и константы получают один и тот же номер

✅ **Автоматическая нумерация** - каждому новому элементу автоматически присваивается уникальный номер

✅ **Форматированный вывод** - таблицы выводятся в красивом табличном формате

✅ **Полная интеграция** - работает с LexicalAnalyzer и TokenType

✅ **Документированный код** - включает примеры использования и полную документацию
