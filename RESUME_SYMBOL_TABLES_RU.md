# 📋 РЕЗЮМЕ: ТАБЛИЦЫ ИМЁН ДЛЯ FOCAL_LEXER

## ❌ ЧТО БЫЛО ТРЕБУЕТСЯ

В модуле `focal_lexer` необходимо было создать таблицы имён (symbol tables), которые собирают информацию о лексемах после лексического анализа.

Требовалось создать таблицы для:
- ✅ Ключевых слов (keywords)
- ✅ Разделителей (delimiters)
- ✅ Операторов (логических/математических)
- ✅ Функций (functions)
- ✅ Идентификаторов (identifiers)
- ✅ Констант (constants)

И написать код их вывода на экран.

---

## ✅ ЧТО БЫЛО СДЕЛАНО

### 1. Создан модуль `symbol_table.py`

Класс `SymbolTable` с полной реализацией всех требуемых таблиц:

#### 🔹 Статические таблицы (предзаполненные):

```python
# Таблица ключевых слов - 11 команд
self.keywords = {'C': {...}, 'A': {...}, 'T': {...}, ...}

# Таблица функций - 10 функций
self.functions = {'FSIN': {...}, 'FCOS': {...}, ...}

# Таблица разделителей - 5 символов
self.delimiters = {',': {...}, ';': {...}, ...}

# Таблица операторов - 6 операторов
self.operators = {'+': {...}, '-': {...}, ...}
```

#### 🔹 Динамические таблицы (заполняются при анализе):

```python
# Таблица идентификаторов
self.identifiers = {}  # заполняется метод add_identifier()

# Таблица констант
self.constants = {}    # заполняется метод add_constant()
```

### 2. API методы

#### Добавление в таблицы:
```python
- add_identifier(name: str, info: str) -> int
- add_constant(value: str, const_type: str) -> int
```

#### Получение информации:
```python
- get_identifier(name: str) -> dict | None
- get_constant(value: str) -> dict | None
```

#### Вывод таблиц:
```python
- print_keywords_table()      # Выводит таблицу ключевых слов
- print_functions_table()     # Выводит таблицу функций
- print_delimiters_table()    # Выводит таблицу разделителей
- print_operators_table()     # Выводит таблицу операторов
- print_identifiers_table()   # Выводит таблицу идентификаторов
- print_constants_table()     # Выводит таблицу констант
- print_all_tables()          # Выводит все таблицы сразу
```
