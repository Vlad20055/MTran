## 🚀 Использование

### Запуск с файлом:
```bash
python3 focal_lexer/main.py focal/test_symbol_table.foc
```

### Запуск примеров:
```bash
python3 focal_lexer/example_usage.py
```

### Из Python программы:
```python
from focal_lexer import SymbolTable, LexicalAnalyzer, TokenType

symbol_table = SymbolTable()
lexer = LexicalAnalyzer(source_code)
tokens = lexer.tokenize()

# Сбор данных в таблицы
for tok in tokens:
    if tok.type == TokenType.IDENTIFIER:
        symbol_table.add_identifier(tok.value)
    elif tok.type == TokenType.NUMBER:
        symbol_table.add_constant(tok.value)

# Вывод таблиц
symbol_table.print_all_tables()
```

## 🔗 Интеграция

Таблицы символов работают со следующими компонентами:

```
LexicalAnalyzer → Token/TokenType → SymbolTable → Таблицы (for display/analysis)
```

Данные текут в одном направлении:
1. Исходный код анализируется LexicalAnalyzer
2. Генерируются токены типа Token с TokenType
3. Информация из токенов добавляется в SymbolTable
4. Таблицы выводятся в красивом формате
